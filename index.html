RespectFace

A no-updates repository of pre-purge Tachiyomi extensions that were signed by Javier Tomás (developer of Tachiyomi). Make sure to star the Codeberg repo.

Guide

One click install (only supported by versions mentioned below):

  • Mihon v0.16.0+
  • Mihon Preview r6442+
  • Tachiyomi v0.15.2+
  • Tachiyomi Preview r6404+
  • TachiyomiSY v1.10.0+
  • TachiyomiSY Preview r539+
  • Aniyomi Preview r7443+
Add to Mihon/Tachiyomi

If you're not using any of the above, download and update extensions from the listing page.

(Optional): To make sure that the extension's apk are signed by the same certificate as original tachiyomi.apk, open apkAnalyzer and check the certificate MD5 of tachiyomi and then of the extension. If both have the same Certificate MD5 as... c4313d134d6b83134103d47a6873cb0f (just check the first 8 characters) ...then its legit safe to use.
